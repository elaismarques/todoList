# Lista de Tarefas Simples (CRUD)

Este é um projeto de lista de tarefas simples desenvolvido em HTML, CSS e JavaScript. 
Ele permite que os usuários criem, leiam, atualizem e excluam tarefas em uma interface de usuário amigável. 
Decidi desenvolvedo porque achei um ótimo projeto de nivel basico para aprender e praticar os conceitos básicos de um sistema CRUD (Criar, Ler, Atualizar, Excluir).
